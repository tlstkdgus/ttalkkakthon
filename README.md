# 🎉 딸깍톤 (Ttalkkakthon)

만우절 기념 MZ 세대 테스트 & 운세 웹 앱입니다.  
React + TypeScript + Vite 기반으로 제작되었습니다.

## 🚀 주요 기능

- 🏠 **홈 화면**: 앱의 메인 페이지로, 다른 기능들로 바로 이동할 수 있는 링크를 제공합니다.
  - **일반 모드**: 재미있고 유쾌한 컨셉의 UI를 제공합니다.
  - **시크릿 모드**: 직장이나 학교에서 몰래 사용할 수 있도록, 마치 업무용 프로그램(엑셀)처럼 보이는 UI 테마로 변경됩니다.

- 📚 **MZ 용어 학습**: 최신 MZ 세대 용어나 밈을 배울 수 있는 기능입니다.
  - 버튼을 누를 때마다 미리 준비된 용어 카드 중 하나가 랜덤으로 표시됩니다.
  - API 호출 없이 가볍고 빠르게 즐길 수 있습니다.

- 🧪 **MZ 지수 테스트**: 여러 개의 퀴즈를 풀며 사용자의 "MZ 지수"를 측정하는 테스트입니다.
  - 각 질문에 대한 답변을 선택하면 다음 질문으로 넘어갑니다.
  - 모든 퀴즈를 완료하면 점수와 함께 재치 있는 결과 설명이 나타납니다.
  - 다시 테스트를 시작할 수 있는 기능도 제공합니다.

- 🔮 **오늘의 운세**: 간단하게 오늘의 운세를 확인하는 기능입니다.
  - 버튼을 누르면 미리 준비된 운세 메시지 중 하나가 랜덤으로 표시됩니다.
  - 시크릿 모드에서는 '오늘의 생존 비결'과 같이 컨셉에 맞는 메시지가 나타납니다.

## 기술 스택

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **State**: Zustand
- **Routing**: React Router DOM

## 개발 환경

```bash
npm install
npm run dev
```
