# 🎉 딸깍톤 (Ttalkkakthon)

만우절 기념 MZ 세대 테스트 & 운세 웹 앱입니다.  
React + TypeScript + Vite 기반으로 제작되었습니다.

## 기능

- 🏠 홈 화면 (일반 / 시크릿 모드)
- 📚 MZ 용어 학습
- 🧪 MZ 지수 테스트
- 🔮 오늘의 운세

## 기술 스택

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **State**: Zustand
- **Routing**: React Router DOM

## 개발 환경

```bash
npm install
npm run dev
```
